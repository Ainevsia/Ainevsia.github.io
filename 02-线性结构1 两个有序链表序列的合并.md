- 好久没用malloc了，连头指针和头结点都分不清楚了。
- 使用头结点可以很方便地解决很多的问题

# 源代码
```C++
List Merge( List L1, List L2 )
{
  List head = (List)malloc(sizeof(struct Node));
  PtrToNode current = head;
  PtrToNode ptr1 = L1->Next;
  PtrToNode ptr2 = L2->Next;
  while (1)
  {
    if (!ptr1) {
      current->Next = ptr2;
      break;
    }
    if (!ptr2) {
      current->Next = ptr1;
      break;
    }
    if (ptr1->Data < ptr2->Data) {
      current->Next = ptr1;
      current = ptr1;
      ptr1 = ptr1->Next;
    }else{
      current->Next = ptr2;
      current = ptr2;
      ptr2 = ptr2->Next;
    }
  }
  L1->Next = NULL;
  L2->Next = NULL;
  return head;
}
```

# 顺便把二分查找的代码也贴上来
```C++
Position BinarySearch( List L, ElementType X )
{
  int start = 1, end = L->Last, mid;
  while (start<=end) {
    mid = (start + end) / 2;
    if (L->Data[mid]<X) {
      start = mid + 1;
    }else if (L->Data[mid]>X) {
      end = mid - 1;
    }else if (L->Data[mid]==X) return mid;
  }
  return NotFound;
}
```
